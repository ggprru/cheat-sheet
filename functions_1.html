Урок 5: Функции в Python

Функции позволяют организовывать код в логические блоки, которые можно многократно использовать.

1. Создание функций

Анатомия функции

def имя_функции(параметры):
    """Документация (docstring)"""
    тело_функции
    return результат  # Необязательно

Примеры функций

# 1. Простая функция
def greet():
    print("Привет, мир!")

# 2. Функция с параметрами
def greet_name(name):
    print(f"Привет, {name}!")

# 3. Функция с возвращаемым значением
def square(x):
    return x * x

# 4. Функция с документацией
def add(a, b):
    """Возвращает сумму двух чисел."""
    return a + b

Важно! Функция не выполняется до момента её вызова.
После def должно быть 4 пробела или 1 таб.

2. Параметры и аргументы

ТипОписаниеПример
ПозиционныеПередаются по порядкуfunc(1, 2)
ИменованныеПередаются по имениfunc(b=2, a=1)

3. Область видимости

x = 10  # Глобальная переменная

def func():
    y = 5  # Локальная переменная
    print(x + y)  # Можно читать глобальные
    
func()
print(y)  # Ошибка! y не существует вне функции

Используйте global для изменения глобальных переменных внутри функции (но лучше избегать)!

Пример решения задачи 16 (сортировка списка)

def bubble_sort(lst):
    """Сортировка пузырьком."""
    n = len(lst)
    for i in range(n-1):
        for j in range(n-i-1):
            if lst[j] > lst[j+1]:
                lst[j], lst[j+1] = lst[j+1], lst[j]
    return lst

numbers = [5, 2, 8, 1, 9]
print(bubble_sort(numbers))  # [1, 2, 5, 8, 9]
