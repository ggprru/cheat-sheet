Словарь терминов Python

Основные понятия для начинающих программистов. Определения и примеры по темам "Переменные" и "Условные выражения".

Тема: Переменные

Переменная
Именованная область памяти для хранения данных. Аналог "коробки" с именем и содержимым.
age = 15 # Переменная age хранит число 15
Тип данных
Характеристика данных, определяющая возможные операции с ними.
type(10) # Возвращает <class 'int'>
int (целое число)
Целые числа без дробной части (положительные, отрицательные и ноль).
count = 42 # Целое число
float (дробное число)
Числа с плавающей точкой (с дробной частью).
pi = 3.1415 # Дробное число
str (строка)
Текстовые данные в кавычках (одинарных или двойных).
name = "Анна" # Строковая переменная
bool (логический тип)
Может принимать только True (истина) или False (ложь).
is_rain = True # Логическая переменная
Оператор присваивания (=)
Присваивает значение переменной.
x = 10 # Присваивание значения
Динамическая типизация
Возможность изменить тип переменной при новом присваивании.
var = 10 # Сначала int
var = "текст" # Теперь str

Тема: Условные выражения

Условный оператор (if)
Выполняет блок кода, если условие истинно.
if age >= 18:
print("Доступ разрешён")
Оператор else
Выполняет блок кода, если условие в if ложно.
if is_rain:
print("Возьми зонт")
else:
print("Зонт не нужен")
Оператор elif
Проверяет дополнительное условие, если предыдущие условия ложны.
if score >= 90:
grade = "A"
elif score >= 80:
grade = "B"
Операторы сравнения
Сравнивают два значения: == (равно), != (не равно), >, <, >=, <=.
10 > 5 # Возвращает True
Логические операторы
Комбинируют условия: and (и), or (или), not (не).
if age >= 18 and has_license:
print("Можно водить")
Вложенные условия
Условия внутри других условий.
if x > 0:
if y > 0:
print("Первая четверть")
Тернарный оператор
Краткая форма записи if-else в одной строке.
status = "Взрослый" if age >= 18 else "Ребёнок"
Булево выражение
Выражение, возвращающее True или False.
is_even = (num % 2 == 0) # True если число чётное
