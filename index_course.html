Roadmap

Тест по курсу Python

Ответьте на следующие вопросы и выполните задания. Удачи!

Часть 1: Теоретические вопросы

  1. Что такое переменная? Приведите пример её создания в Python.
  2. Какие типы данных существуют в Python? Перечислите их.
  3. Объясните разницу между `if` и `elif`.
  4. Как работает цикл `for`? Приведите пример.
  5. Что такое функция? Чем она отличается от обычного кода?
  6. Для чего используются списки и словари? Приведите примеры.
  7. Как записать данные в файл в Python?
  8. Что такое класс в ООП? Назовите его основные компоненты.

Часть 2: Практические задания

Раздел 1: Переменные и типы данных

  1. Напишите программу, которая запрашивает имя и возраст пользователя, а затем выводит: "Привет, [Имя]! Тебе [Возраст] лет."
  2. Создайте переменные для хранения текущей температуры в градусах Цельсия и преобразуйте её в градусы Фаренгейта по формуле: F = C * 9/5 + 32.

Раздел 2: Условные операторы

  1. Напишите программу, которая проверяет, является ли число, введённое пользователем, чётным.
  2. Напишите программу, которая запрашивает у пользователя оценку (0–100) и выводит соответствующую категорию:
    • 90–100: "Отлично"
    • 70–89: "Хорошо"
    • 50–69: "Удовлетворительно"
    • 0–49: "Плохо"

Раздел 3: Циклы

  1. Напишите программу, которая выводит таблицу умножения для числа, введённого пользователем.
  2. Напишите программу, которая вычисляет сумму чисел от 1 до 100.

Раздел 4: Списки и словари

  1. Создайте список из пяти любимых фруктов, добавьте в список ещё один фрукт и выведите его длину.
  2. Создайте словарь с данными об учебном предмете (название, преподаватель, часы в неделю). Выведите все ключи и значения.

Раздел 5: Функции

  1. Напишите функцию, которая принимает два числа и возвращает их сумму.
  2. Напишите функцию, которая принимает строку и возвращает её в обратном порядке (например, "Привет" → "тевирП").

Раздел 6: Файлы

  1. Напишите программу, которая создаёт файл `data.txt` и записывает в него: "Это тестовый файл."
  2. Напишите программу, которая читает данные из файла `data.txt` и выводит их на экран.

Раздел 7: ООП

  1. Создайте класс `Student` с атрибутами `name` и `age`. Добавьте метод `introduce`, который выводит: "Привет, меня зовут [name], и мне [age] лет."
  2. Наследуйте класс `HighSchoolStudent` от `Student`, добавьте атрибут `grade` (класс обучения) и метод для вывода этого атрибута.

Часть 3: Сложные задания

  1. Напишите программу, которая генерирует случайное число от 1 до 100 и просит пользователя его угадать. После каждой попытки укажите, было ли число больше или меньше.
  2. Напишите программу, которая принимает строку от пользователя и считает количество гласных в ней.

Максимальный балл: 100 очков

Руководство по оценке:

  • 90–100: Отлично
  • 75–89: Хорошо
  • 50–74: Удовлетворительно
  • Менее 50: Требуется улучшение
