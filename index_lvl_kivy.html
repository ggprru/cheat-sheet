Kivy UI Виджеты Cheat Sheet

1. Установка

pip install kivy

2. Основные UI-виджеты в Kivy

2.1. Label (Текстовый виджет)

from kivy.uix.label import Label
label = Label(text='Привет, Kivy!', font_size=24, color=(1, 0, 0, 1))
  • text: текстовая строка
  • font_size: размер шрифта
  • color: цвет в формате RGBA

2.2. Button (Кнопка)

from kivy.uix.button import Button
button = Button(text='Нажми меня', size_hint=(.5, .5))
  • text: текст на кнопке
  • size_hint: масштабирование относительно родительского контейнера
  • on_press: обработка нажатия

2.3. TextInput (Поле ввода текста)

from kivy.uix.textinput import TextInput
text_input = TextInput(text='Введите что-то', multiline=False)
  • text: начальный текст
  • multiline: возможность ввода нескольких строк
  • password: скрытый ввод

2.4. CheckBox (Флажок)

from kivy.uix.checkbox import CheckBox
checkbox = CheckBox(active=True)
  • active: состояние (True/False)

2.5. Slider (Ползунок)

from kivy.uix.slider import Slider
slider = Slider(min=0, max=100, value=50)
  • min: минимальное значение
  • max: максимальное значение
  • value: текущее значение

2.6. Switch (Переключатель)

from kivy.uix.switch import Switch
switch = Switch(active=True)
  • active: включено/выключено

2.7. ProgressBar (Индикатор прогресса)

from kivy.uix.progressbar import ProgressBar
progress_bar = ProgressBar(max=100, value=40)
  • max: максимальное значение
  • value: текущее значение

2.8. Image (Изображение)

from kivy.uix.image import Image
image = Image(source='logo.png')
  • source: путь к изображению

3. Контейнеры (Layouts)

3.1. BoxLayout (Горизонтальный и вертикальный контейнер)

from kivy.uix.boxlayout import BoxLayout
layout = BoxLayout(orientation='vertical')
  • orientation: 'horizontal' или 'vertical'

3.2. GridLayout (Таблица)

from kivy.uix.gridlayout import GridLayout
layout = GridLayout(cols=2)
  • cols: количество колонок
  • rows: количество строк

3.3. AnchorLayout (Прикрепленный макет)

from kivy.uix.anchorlayout import AnchorLayout
layout = AnchorLayout(anchor_x='center', anchor_y='top')
  • anchor_x: выравнивание по горизонтали ('left', 'center', 'right')
  • anchor_y: выравнивание по вертикали ('top', 'center', 'bottom')

3.4. FloatLayout (Свободное расположение)

from kivy.uix.floatlayout import FloatLayout
layout = FloatLayout()
  • size_hint: относительное позиционирование
  • pos_hint: задает положение в процентах

4. Обработка событий

def on_button_click(instance):
    print('Кнопка нажата!')

button = Button(text='Кликни меня')
button.bind(on_press=on_button_click)

5. Использование KV Language для интерфейса

myapp.kv:

BoxLayout:
    orientation: 'vertical'
    Button:
        text: 'Кнопка 1'
    Button:
        text: 'Кнопка 2'

main.py:

from kivy.app import App
from kivy.lang import Builder

class MyApp(App):
    def build(self):
        return Builder.load_file('myapp.kv')

MyApp().run()
