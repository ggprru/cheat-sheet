Условные выражения в Python

Условные операторы помогают программам "принимать решения" в зависимости от условий.

1. Что такое условие?

Это проверка, которая возвращает True (правда) или False (ложь).

Пример из жизни:

«Если на улице дождь (is_rain = True), то взять зонт»

2. Операторы сравнения

ОператорПримерОписание
==5 == 5Равно
!=3 != 5Не равно
>10 > 5Больше
<3 < 7Меньше
>=age >= 18Больше или равно
<=temp <= 0Меньше или равно

3. Конструкция if

if условие:
    # Код, который выполнится если условие True
    print("Условие верно!")

Проверка возраста:

age = 15
if age < 18:
    print("Ты ещё не взрослый!")

4. Добавляем else

if условие:
    print("Условие верно")
else:
    print("Условие неверно")

Проверка пароля:

password = "qwerty123"
if password == "12345":
    print("Пароль верный!")
else:
    print("Неверный пароль!")

5. Множественные условия (elif)

if x > 0:
    print("Положительное")
elif x < 0:
    print("Отрицательное")
else:
    print("Ноль!")

Важно!

  • Отступы (4 пробела) обязательны!
  • После условия ставится :
  • elif = "иначе если"

Практические задания

Уровень 1 (Начальный)

  1. Дано num = 10. Проверьте, равно ли число 5
  2. Создайте переменную is_rain = True. Если дождь, выведите "Возьми зонт!"
  3. Проверьте, что 100 > 50 (выведите результат)

Уровень 2 (Средний)

  1. Дано age = 17. Выведите "Ребёнок" (если < 18) или "Взрослый"
  2. Проверьте, является ли число num = 7 чётным (% 2 == 0)
  3. Для temp = -5 выведите "Лёд" (если ≤0) или "Вода"

Уровень 3 (Продвинутый)

  1. Даны login = "admin" и password = "123". Проверьте их корректность
  2. Для числа num = 15 проверьте: кратно ли оно 3 и 5 одновременно
  3. Определите время суток (утро/день/вечер/ночь) по hour = 14

Пример решения

Решение задания №9 (время суток):

hour = 14
if 6 <= hour < 12:
    print("Утро")
elif 12 <= hour < 18:
    print("День")
elif 18 <= hour < 23:
    print("Вечер")
else:
    print("Ночь")
