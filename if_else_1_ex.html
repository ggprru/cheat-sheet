Дополнительные задания: Условные выражения

50 задач для отработки условий if/elif/else с постепенным увеличением сложности.

Уровень 1: Базовые условия (15 заданий)

  1. Дано число num = 10. Проверьте, положительное ли оно
  2. Создайте переменную is_rain = True. Если дождь, выведите "Возьми зонт"
  3. Проверьте, что 15 > 20 (выведите результат)
  4. Дано age = 17. Выведите "Подросток" если возраст 13-19 лет
  5. Для temp = -3 выведите "Минус" если температура < 0
  6. Проверьте, является ли num = 4 чётным числом
  7. Дано password = "qwerty". Проверьте, равен ли он "12345"
  8. Для score = 75 выведите "Сдал" если score ≥ 60
  9. Создайте is_weekend = False. Если не выходной, выведите "Иди в школу"
  10. Проверьте, что строка s = "Python" не равна "python" (учтите регистр)
  11. Дано x = 5, y = 5. Проверьте, что x равен y
  12. Для year = 2023 проверьте, что он не равен 2020
  13. Создайте is_day = True. Если день, выведите "Солнце светит"
  14. Проверьте, что длина строки word = "Hello" больше 3
  15. Дано a = 7, b = 10. Проверьте, что a не равно b

Уровень 2: Комбинированные условия (20 заданий)

  1. Дано age = 20. Выведите "Ребёнок" (<12), "Подросток" (12-17) или "Взрослый" (18+)
  2. Для temp = 25 выведите "Холодно" (<10), "Тепло" (10-20) или "Жарко" (>20)
  3. Проверьте, является ли num = 15 чётным и больше 10 одновременно
  4. Даны login = "admin", password = "qwerty". Проверьте их корректность
  5. Для score = 87 выведите оценку: A (≥90), B (≥80), C (≥70), D (<70)
  6. Создайте time = 14. Определите "Утро" (6-11), "День" (12-17), "Вечер" (18-23), "Ночь"
  7. Проверьте, является ли year = 2024 високосным годом
  8. Дано num = 30. Проверьте, кратно ли оно 5 или 10
  9. Для a = 10, b = 20, c = 15 найдите максимальное число
  10. Создайте word = "топот". Проверьте, является ли оно палиндромом
  11. Даны 3 числа. Определите, все ли они чётные
  12. Для price = 100, discount = 30 вычислите итоговую цену (если скидка >20%, иначе цена без скидки)
  13. Проверьте, является ли num = 36 квадратом целого числа
  14. Дано s = "Python". Проверьте, начинается ли оно с "Py" и заканчивается на "on"
  15. Для a = 3, b = 4, c = 5 проверьте, может ли это быть треугольник
  16. Создайте age = 16, has_parents = True. Можно ли смотреть фильм 18+ (нет, но если есть родители - можно)
  17. Проверьте, является ли num = 999 трёхзначным числом
  18. Даны x = 5, y = 10. Какое число ближе к 8?
  19. Для day = "сб" определите, рабочий это день (пн-пт) или выходной
  20. Создайте a = 15, b = 10. Поменяйте значения если a > b

Уровень 3: Сложные условия (15 заданий)

  1. Для year = 2023, month = 2 определите количество дней в месяце (учтите високосность)
  2. Даны 3 числа. Определите, могут ли они быть сторонами прямоугольного треугольника
  3. Создайте time = 3665. Переведите в часы:минуты:секунды (формат HH:MM:SS)
  4. Проверьте, является ли ip = "192.168.1.1" корректным IPv4 адресом (4 числа 0-255 через точки)
  5. Для date = "31.02.2023" проверьте, является ли дата корректной (учитывая месяцы и високосные годы)
  6. Даны a = 5, b = 3, c = 8. Упорядочьте их по возрастанию
  7. Создайте color = "#FF00FF". Определите, является ли он красным (первые 2 символа FF) или синим (последние 2 FF)
  8. Проверьте, можно ли из букв слова word = "привет" составить target = "ветер"
  9. Для num = 12321 проверьте, является ли оно палиндромом (читается одинаково слева направо)
  10. Создайте text = "I love Python". Зашифруйте его (сдвиг букв на +1: J mpwf Qzuipo)
  11. Даны координаты шахматной клетки pos = "e4". Определите её цвет (черная/белая)
  12. Для треугольника со сторонами a, b, c определите его тип (равносторонний, равнобедренный, разносторонний)
  13. Проверьте, является ли строка s = "{[()]}" корректной последовательностью скобок
  14. Создайте калькулятор ИМТ: по weight и height определите категорию (недостаток, норма, избыток)
  15. Реализуйте простой калькулятор (+, -, *, /) для двух чисел с проверкой деления на ноль

Пример решения задачи 36 (дни в месяце)

year = 2023
month = 2

if month == 2:
    # Проверка на високосный год
    if (year % 400 == 0) or (year % 100 != 0 and year % 4 == 0):
        days = 29
    else:
        days = 28
elif month in [4, 6, 9, 11]:
    days = 30
else:
    days = 31

print(f"В месяце {days} дней")

Для решения сложных задач может потребоваться изучение следующих тем: списки, циклы, функции. Рекомендуется разбирать по 5-10 заданий за урок.
