Урок 4: Циклы в Python

Циклы позволяют повторять блок кода многократно. В Python есть два основных типа циклов: for и while.

1. Цикл for

Схема работы цикла for

Используется для перебора элементов последовательности:

# Перебор списка
fruits = ['яблоко', 'банан', 'апельсин']
for fruit in fruits:
    print(fruit)

Важно! В Python цикл for - это for-each, он перебирает элементы, а не работает с индексами как в некоторых других языках.

Функция range()

range() - встроенная функция Python для генерации последовательностей чисел. Используется преимущественно в циклах.

range() создает "виртуальную" последовательность, не занимая память

1. Основные формы записи

ФормаОписаниеПримерРезультат
range(stop)От 0 до stop-1range(5)0,1,2,3,4
range(start, stop)От start до stop-1range(2,6)2,3,4,5
range(start, stop, step)От start до stop-1 с шагом steprange(1,10,2)1,3,5,7,9

2. Подробное объяснение параметров

# Параметр stop (обязательный)
for i in range(5):     # 0,1,2,3,4
    print(i)

# Параметры start и stop
for i in range(2, 6):  # 2,3,4,5
    print(i)

# Параметр step (шаг)
for i in range(1, 10, 2):  # 1,3,5,7,9
    print(i)

# Отрицательный шаг
for i in range(10, 0, -1):  # 10,9,8,...,1
    print(i)

Важно! range() не включает конечное значение (stop). Это "полуинтервал" [start, stop).

3. Продвинутые возможности

# Создание списка из range
numbers = list(range(5))  # [0,1,2,3,4]

# Использование в генераторах списков
squares = [x**2 for x in range(10)]

# Индексация range объекта
r = range(10, 20, 2)
print(r[2])    # 14 (третий элемент)
print(r[-1])   # 18 (последний элемент)

# Проверка вхождения
print(5 in range(10))   # True
print(15 in range(1,10)) # False

# Длина range
print(len(range(5, 50, 3)))  # 15 элементов

4. Оптимизация работы с range

Range не создает список в памяти - это "ленивый" генератор. Примеры эффективного использования:

# Эффективный перебор большого диапазона
for i in range(1, 1000000):  # Не расходует память
    if i % 100000 == 0:
        print(i)

# Сравнение с реальным списком (неэффективно)
# list(range(1000000)) - создаст реальный список в памяти!

5. Практические примеры

# Обратный отсчет
for i in range(10, 0, -1):
    print(f"Осталось {i} секунд...")
print("Пуск!")

# Обработка каждого N-го элемента
data = ['a','b','c','d','e','f','g','h']
for i in range(0, len(data), 2):  # Каждый второй
    print(data[i])

# Создание координатной сетки
for x in range(5):
    for y in range(5):
        print(f"({x},{y})", end=' ')
    print()

Частые ошибки

  • Забывают, что stop не включается: range(5) → 0,1,2,3,4
  • Пустой диапазон при неправильном шаге: range(5, 1) → пусто
  • Использование float: range(1.5, 5.5) → TypeError

2. Цикл while

Схема работы цикла while

Выполняется, пока условие истинно:

# Простой пример
count = 0
while count < 5:
    print(count)
    count += 1

# Бесконечный цикл с break
while True:
    answer = input("Введите 'выход' для завершения: ")
    if answer.lower() == 'выход':
        break
    print("Вы ввели:", answer)

3. Управление циклами

КомандаОписаниеПример
breakПолностью прерывает циклwhile True: break
continueПереходит к следующей итерацииfor i in range(5): if i == 2: continue
elseВыполняется, если цикл завершился без breakfor i in range(3): ... else: print("Готово")

Пример решения задачи 16 (проверка на простое число)

num = 17
is_prime = True

if num > 1:
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            is_prime = False
            break

print(f"Число {num} {'простое' if is_prime else 'составное'}")
