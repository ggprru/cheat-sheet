Введение в Python

Python — это высокоуровневый язык программирования с простым синтаксисом, который идеально подходит для начинающих. Он используется в веб-разработке, анализе данных, автоматизации, создании игр и других областях.

Примеры простых программ на Python:

# Программа, которая выводит текст
print("Hello, World!")

# Программа, которая запрашивает имя и приветствует пользователя
name = input("Введите ваше имя: ")
print("Привет, " + name + "!")
