Рекурсия

Рекурсия — это метод, при котором функция вызывает саму себя для решения подзадач. Важно иметь базовый случай, чтобы избежать бесконечной рекурсии.

Пример функции для расчета факториала с рекурсией:

def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Вывод: 120

Пример функции для вычисления числа Фибоначчи:

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(7))  # Вывод: 13
