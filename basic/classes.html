Классы и объекты

Классы — это шаблоны для создания объектов, а объекты — это экземпляры классов. Классы позволяют организовать код и создавать структуры для хранения данных и методов.

Пример создания класса и объекта:

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} говорит: Гав-гав!")

dog = Dog("Бобик", 3)
dog.bark()  # Вывод: Бобик говорит: Гав-гав!
