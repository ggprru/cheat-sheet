Лямбда-функции

Лямбда-функции — это анонимные функции, которые создаются с помощью ключевого слова lambda. Они позволяют создавать небольшие однострочные функции без имени.

Пример лямбда-функции:

multiply = lambda x, y: x * y
print(multiply(3, 4))  # Вывод: 12

Использование лямбда-функций с map и filter:

# Применение функции к каждому элементу
numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Вывод: [1, 4, 9, 16]

# Фильтрация элементов
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # Вывод: [2, 4]
