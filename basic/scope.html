Область Видимости Переменных

Область видимости переменных определяет, где переменная доступна в коде. В Python есть локальная и глобальная области видимости.

Пример локальной переменной:

def my_function():
    x = 10  # Локальная переменная
    print(x)

my_function()
# print(x)  # Ошибка, так как x не доступна вне функции

Пример глобальной переменной:

x = 5  # Глобальная переменная

def my_function():
    print(x)

my_function()  # Вывод: 5
