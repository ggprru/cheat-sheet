Словари

Словарь — это неупорядоченная коллекция пар "ключ-значение". Каждый элемент в словаре имеет уникальный ключ, который используется для доступа к значению.

Пример работы со словарями:

person = {"имя": "Алиса", "возраст": 25, "город": "Москва"}
print(person["имя"])  # Выводит "Алиса"

Основные методы работы со словарями:

  • keys() — получение всех ключей
  • values() — получение всех значений
  • items() — получение всех пар "ключ-значение"
  • get() — получение значения по ключу

Пример использования методов:

print(person.keys())    # Все ключи
print(person.values())  # Все значения
print(person.get("город"))  # Значение ключа "город"
