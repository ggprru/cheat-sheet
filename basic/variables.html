Переменные и Типы Данных

Переменная — это контейнер для хранения данных. В Python переменные создаются при присвоении значения, и их тип определяется автоматически.

Пример:

name = "Алиса"  # Строка (str)
age = 25        # Целое число (int)
height = 1.75   # Число с плавающей точкой (float)
is_student = True  # Логическое значение (bool)

Основные типы данных:

  • str: строки, текстовая информация
  • int: целые числа
  • float: числа с плавающей точкой
  • bool: логические значения (True или False)

Преобразование типов:

# Преобразование строки в число
number = "10"
converted_number = int(number)  # теперь это целое число 10
