Списки

Список — это упорядоченная коллекция элементов, которая позволяет хранить несколько значений в одной переменной. Списки изменяемы, что означает, что их можно изменять после создания.

Пример работы со списком:

fruits = ["яблоко", "банан", "вишня"]
print(fruits[0])  # "яблоко"
fruits.append("апельсин")  # Добавляет "апельсин" в конец списка
print(fruits)

Основные методы работы со списками:

  • append() — добавление элемента в конец списка
  • remove() — удаление элемента из списка
  • sort() — сортировка списка
  • len() — длина списка

Пример работы с методами:

fruits = ["яблоко", "банан", "вишня"]
fruits.append("апельсин")  # Добавить элемент
fruits.remove("банан")     # Удалить элемент
fruits.sort()              # Сортировать список
print(len(fruits))         # Количество элементов
