Функции

Функции — это блоки кода, которые можно вызывать многократно. Функции помогают структурировать и организовать код.

Пример функции:

def greet(name):
    print(f"Привет, {name}!")

greet("Алиса")  # Вывод: Привет, Алиса!

Функции с возвращаемым значением:

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Вывод: 8
