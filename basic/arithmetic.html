Арифметические Операции

Python поддерживает стандартные арифметические операции: сложение, вычитание, умножение и деление.

Пример:

a = 10
b = 3

# Базовые арифметические операции
print(a + b)  # Сложение
print(a - b)  # Вычитание
print(a * b)  # Умножение
print(a / b)  # Деление (результат — float)

Дополнительные операции:

  • // — целочисленное деление (отбрасывает остаток)
  • % — остаток от деления
  • ** — возведение в степень

Пример с дополнительными операциями:

print(a // b)  # Целочисленное деление (3)
print(a % b)   # Остаток от деления (1)
print(a ** b)  # Возведение в степень (1000)
