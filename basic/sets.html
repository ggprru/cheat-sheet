Множества

Множество — это неупорядоченная коллекция уникальных элементов. Множества используются, когда нужно гарантировать, что элементы не будут дублироваться.

Пример работы с множествами:

fruits = {"яблоко", "банан", "вишня"}
fruits.add("апельсин")  # Добавление элемента
print(fruits)

Основные операции с множествами:

  • add() — добавление элемента
  • remove() — удаление элемента
  • union() — объединение множеств
  • intersection() — пересечение множеств

Пример объединения и пересечения:

set1 = {1, 2, 3}
set2 = {2, 3, 4}

print(set1.union(set2))         # {1, 2, 3, 4}
print(set1.intersection(set2))  # {2, 3}
