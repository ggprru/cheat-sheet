Наследование и полиморфизм

Наследование позволяет одному классу (подклассу) наследовать методы и свойства другого класса (родительского класса). Полиморфизм позволяет методам вести себя по-разному в зависимости от класса объекта.

Пример наследования:

class Animal:
    def speak(self):
        print("Животное издает звук")

class Dog(Animal):
    def speak(self):
        print("Собака говорит: Гав-гав")

dog = Dog()
dog.speak()  # Вывод: Собака говорит: Гав-гав

Полиморфизм в действии:

animals = [Dog(), Animal()]

for animal in animals:
    animal.speak()
# Вывод:
# Собака говорит: Гав-гав
# Животное издает звук
