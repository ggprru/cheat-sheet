Параметры и Возвращаемые Значения

Функции могут принимать параметры и возвращать значения. Это позволяет сделать функции гибкими и многоразовыми.

Пример функции с параметрами:

def multiply(a, b):
    return a * b

print(multiply(4, 5))  # Вывод: 20

Функции с несколькими возвращаемыми значениями:

def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder

result, rem = divide(10, 3)
print(result, rem)  # Вывод: 3 1
