Циклы

Циклы позволяют выполнять блок кода многократно, что полезно при обработке элементов коллекций или выполнении повторяющихся операций.

Цикл for:

fruits = ["яблоко", "банан", "вишня"]
for fruit in fruits:
    print(fruit)

Цикл while:

count = 0
while count < 5:
    print(count)
    count += 1
