Модули и пакеты

Модули — это файлы Python, содержащие функции, классы и переменные. Они позволяют организовать и структурировать код. Пакеты — это наборы модулей, которые можно использовать как единое целое.

Импорт модуля:

# Импорт стандартного модуля math
import math

print(math.sqrt(16))  # Вывод: 4.0

Создание и использование пользовательского модуля:

# В файле my_module.py
def greet(name):
    return f"Привет, {name}!"

# В основном файле
import my_module

print(my_module.greet("Алиса"))  # Вывод: Привет, Алиса!
